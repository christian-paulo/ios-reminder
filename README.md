
# iOS Reminder SPM

This program generate a reminder with Title and Hour/Min. I used the Event Kit Framework
for Apple
## Install

Install ios-remidner with spm

```bash
git clone https://github.com/christianpauloios/ios-reminder
cd ios-reminder
swift run remindercp
```
